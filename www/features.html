Features

  • Raw TCP handling — Connections managed directly with net.Listener and net.Conn
  • HTTP/1.1 request parsing — Method, URL, headers, and body extracted from raw bytes
  • Static file serving — Files served from the filesystem with automatic directory index resolution
  • Content-type detection — Responses set the correct type for .html, .css, .js, .json, and .txt files
  • Concurrent connections — Each client is handled in its own goroutine
  • Zero dependencies — Only the Go standard library is used in production
